# 🏠 House Price Prediction – Regression Model

## 📌 Project Overview
This project focuses on predicting house prices using the Boston Housing dataset.

The objective was to:

- Perform data preprocessing  
- Handle missing values  
- Build a regression model  
- Evaluate performance using RMSE  
- Visualize predicted vs actual values  

---

## 🔍 Analysis Performed

### Data Preprocessing
- Handled missing values using median and mode  
- Verified dataset structure and cleaned data  

### Model Building
- Used Linear Regression model  
- Split dataset into training and testing sets  

### Model Evaluation
- Evaluated model using RMSE  
- Achieved RMSE value of **4.99**  

### Visualization
- Scatter plot of actual vs predicted values  
- Regression fit visualization  

---

## 📊 Key Insights
- The model shows a good relationship between features and house prices  
- Predictions are reasonably close to actual values  
- RMSE value indicates acceptable prediction accuracy  
- Some prediction errors exist due to model simplicity  
- Model performance can be improved using advanced techniques
